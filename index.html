About This Project

This website is created as an academic project using HTML and CSS, inspired by the visual style of old cinema. The aim of the project is not only to present information but also to design an immersive cinematic experience.

Minimal JavaScript is used only for navigation behavior, while the focus remains on layout, hover interactions, responsiveness, and visual storytelling through design.
